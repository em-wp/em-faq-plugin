List of FAQs with open/close answer feature added to page with shortcode.

== Description ==

Uses tinymce on each page and post edit pages to add FAQ to that page with a shortcode. Adds google serp feature with strucutered data through json-ld format.

Features:
* No javascript.
* Inline css only.
* Every FAQ is page specific.
* Responsive.
* Multiple Design choices.
* Add colors through shortcode.
 
== Frequently Asked Questions ==
 
= How do I use this plugin? =

Add a new FAQ element on the page/post edit page and add the FAQ list to the page/post with the shortcode [faq].
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin. 
Note: The FAQ data will not be deleted.
